# media-library-website
A website includes informations about some movies, series and books (in Turkish).

# Technologies Used
The following technologies were used in developing this project:

* HTML5 Semantic structure and content creation.
* CSS3: Design, responsive (mobile-friendly) appearance and animations.
* JavaScript: Dynamic content management and interactions.
* PWA (Progressive Web App): Mobile application-like experience with manifest.json and sw.js (Service Worker).

# Features
* Mobile-friendly (Responsive) design.
* Can be added to the home screen thanks to PWA support.
* Dark mode support.
* Data storage (Local Storage).
* Animation feature.

live (demo) link: https://evra-is-coding.github.io/media-library-website/
